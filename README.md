# SignLanguageBackend

A Flask-based backend for a Sign Language application.

---

## Prerequisites

- **Python**: 3.9.13  
- **pip**: 22.0.4  
- (Optional) Virtual environment tool (`venv`)

---

## Setup & Installation

-  install the requirements.txt 


## After Installation 
-run the code with (python app.py) In terminal

## once its run 

- http://localhost:5000/  open in the tab
